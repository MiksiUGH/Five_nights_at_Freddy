# Five Nights at Freddy's (фанатская реализация)

Фанатская версия культовой игры в жанре survival horror с видом сверху. Игроку предстоит выживать в замкнутом пространстве, уклоняясь от аниматроников, которые активируются и начинают охоту.

## Описание

Вы — ночной сторож, который должен продержаться как можно дольше, перемещаясь по локации и прячась в укрытиях. Враги — аниматроники Бонни, Чика, Фокси и Фредди — обладают уникальными моделями поведения. Цель игры — не попасться ни одному из них.

## Управление

| Клавиша | Действие |
|---------|----------|
| W / S / A / D | Движение игрока |
| Пробел | Вход в укрытие (при нахождении рядом с объектом укрытия) |
| Пробел (в укрытии) | Выход из укрытия |
| E | Взаимодействие с дверями (телепорт на другую сторону) |
| ESC | Пауза / открытие меню паузы |

## Особенности геймплея

- **Динамическое освещение** — вокруг игрока светлый круг, всё остальное затемнено.
- **Укрытия** — игрок может спрятаться, но не дольше 7 секунд, после чего персонаж автоматически выходит.
- **Аниматроники активируются случайно** — после активации они начинают преследование или применяют особые механики.
- **Тряска камеры** — если стоять на месте слишком долго, начинается тряска, затем скример.
- **Сохранение результатов** — время выживания записывается в базу данных SQLite, лучшие результаты можно посмотреть в меню статистики.

## Аниматроники

### Бонни
- Изначально неактивен, каждые 10 секунд может активироваться с вероятностью 80%.
- Патрулирует случайным образом, при виде игрока переходит в погоню.
- В погоне скорость увеличивается, анимация ускоряется.
- Если игрок прячется в укрытие, теряет цель и возвращается к патрулю.

### Чика
- При активации исчезает, на карте появляется кекс.
- Кекс перемещается каждые 15 секунд.
- Если игрок касается кекса — мгновенная смерть.

### Фокси
- Активируется с вероятностью 60% каждые 20 секунд.
- После активации начинает красться, меняя текстуры (4 стадии).
- Если достиг последней стадии, а игрок не в укрытии — бросается в погоню.
- В погоне движется быстро, игнорирует невидимость.

### Фредди
- Активируется с вероятностью 70% каждые 20 секунд.
- После активации исчезает и начинает следить за неподвижностью игрока.
- Если игрок стоит на месте в пределах квадрата 500×500 пикселей дольше 20 секунд, начинается нарастающая тряска камеры.
- Через 5 секунд тряски — скример и смерть.
- При перемещении игрока за пределы зоны таймер сбрасывается.

## Требования для запуска

- Python 3.8 или выше
- Библиотека `arcade`
- Поддержка OpenGL

## Установка и запуск

1. Склонируйте репозиторий или распакуйте архив с игрой.
2. Перейдите в папку проекта.
3. Установите зависимости:
   ```bash
   pip install arcade
   ```
4. Запустите игру:
   ```bash
   python window.py
   ```

## Используемые технологии

- **Python** — основной язык программирования.
- **Arcade** — библиотека для создания 2D-игр.
- **SQLite3** — хранение результатов.
- **Pyglet** — подсистема OpenGL (используется через Arcade).
